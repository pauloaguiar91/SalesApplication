SalesApplication
================

Developed at Algonquin College by

Paulo Aguiar
Tania Rossi
Gregory Nikitin
Shengwei He

This sales application was all contained in C# forms. 

These forms included the functionality to:

Add new Customers, Employees, Suppliers, Shippers and Categories of items into a local Access database.
View products on display, and select them for purchase.
Confirm your purchase for a confirmation form.

(2012)
